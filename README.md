**Iris Species Detector**

A deep learning mobile application to classify Iris flower species using a Multi-Layer Perceptron (MLP).

**Architecture**

Input Layer: 4 Features (Sepal/Petal dimensions)

Hidden Layer 1: 10 Neurons (ReLU)

Hidden Layer 2: 20 Neurons (ReLU)


Output Layer: 3 Neurons (Softmax)

**Files**

train_model.py: Python script for training.

iris_classifier_v2.tflite: Trained model file.

App.js: React Native application code.

**How to Use**

Enter Sepal Length and Width.

Enter Petal Length and Width.

Press "Analyze Flower" to get the species prediction.
